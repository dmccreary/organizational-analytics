# References: Reporting and Dashboards

1. [Dashboard (business)](https://en.wikipedia.org/wiki/Dashboard_(business)) - Wikipedia - Overview of business dashboard design principles including key performance indicators, visual hierarchy, and real-time data presentation. Core reference for the executive and operational dashboard patterns in this chapter.

2. [Data Visualization](https://en.wikipedia.org/wiki/Data_and_information_visualization) - Wikipedia - Covers visualization theory, chart types, perceptual principles, and interactive techniques. Essential background for translating graph metrics into clear, actionable visual presentations for leadership.

3. [Anomaly Detection](https://en.wikipedia.org/wiki/Anomaly_detection) - Wikipedia - Statistical and machine learning methods for identifying unusual patterns in data streams. Applicable to the alerting systems that monitor organizational graph metrics for significant deviations from baselines.

4. The Visual Display of Quantitative Information (2nd Edition) - Edward Tufte - Graphics Press (2001) - The classic reference on data visualization principles including data-ink ratio, chartjunk elimination, and small multiples. Directly applicable to designing clean, effective organizational analytics dashboards.

5. Storytelling with Data: A Data Visualization Guide for Business Professionals - Cole Nussbaumer Knaflic - Wiley (2015) - Practical guide to communicating data insights to business audiences through context, visual design, and narrative structure. Relevant to presenting organizational analytics findings to leadership.

6. [Information Dashboard Design](https://en.wikipedia.org/wiki/Dashboard_(business)#Design) - Wikipedia - Specific design guidelines for dashboard layout, color usage, interaction patterns, and progressive disclosure applicable to organizational analytics reporting interfaces.

7. [Key Performance Indicator](https://en.wikipedia.org/wiki/Performance_indicator) - Wikipedia - Frameworks for defining and measuring KPIs that drive organizational decision-making. Provides context for selecting which graph metrics (centrality, connectivity, community health) to surface on dashboards.

8. [Business Intelligence](https://en.wikipedia.org/wiki/Business_intelligence) - Wikipedia - Overview of BI tools, OLAP cubes, and reporting architectures. Provides context for how organizational graph analytics integrates with existing enterprise reporting infrastructure.

9. [D3.js Data-Driven Documents](https://d3js.org/) - Observable - Documentation for the JavaScript library commonly used to create custom, interactive network visualizations and dashboards in web-based organizational analytics applications.

10. [Apache Superset](https://superset.apache.org/) - Apache Foundation - Open-source business intelligence platform for creating interactive dashboards and visualizations. Example of modern BI tooling that can connect to graph database query results for organizational reporting.
